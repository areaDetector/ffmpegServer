ffmpegServer EPICS Support Module

Introduction

ffmpegServer is a windows and linux areaDetector plugin wrapping the ffmpeg libraries that provides 2 functions:

  • ffmpegStream: Compression into an mjpg stream which is made available over http.
  • ffmpegFile: Compression to disk into any file format that ffmpeg supports.

Please click on one of the links above for documentation on the plugin.

Dependencies

ffmpegServer requires the following:

  • areaDetector - tested with version 1-9
  • ffmpeg - tested with version 4.0

Install instructions

First of all, please read the information on ffmpeg legal information to check that your usage of ffmpeg is legal in your country. As this module is just a wrapper to ffmpeg, it is not distributed with ffmpeg, so you need to download and compile ffmpeg in order to use the module. The following steps need to be accomplished, either using the bash script install.sh in the root of the module, or by doing them manually:

  • (Linux only) Download and unzip the source to vendor/ffmpeg
  • (Linux only) Download and unzip yasm to vendor/yasm
  • (Win32 only) Download and unzip win32 libraries to vendor/ffmpeg-win32-shared
  • (Win32 only) Download and unzip win32 devel libraries to vendor/ffmpeg-win32-shared
  • (Win64 only) Download and unzip win64 libraries to vendor/ffmpeg-win64-shared
  • (Win64 only) Download and unzip win64 devel libraries to vendor/ffmpeg-win64-shared

You can now edit configure/RELEASE and type make to build
