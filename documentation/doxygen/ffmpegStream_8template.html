Detailed Description

ffmpegServer contains a very simple http server based on nullhttpd. If a file named <port>.jpg is requested for any ffmpegStream object named <port>, then it will return the last NDArray received by the object compressed to jpeg. If <port>.mjpg is requested then it will return an mjpg stream over http to the client. Otherwise, it will return an index page listing all the available streams.

Setup

  • In the database, an instance of NDPluginBase is required, followed by an instance of this template.
  • In the startup scripts, call ffmpegServerConfigure() once per IOC to setup the http server, then call ffmpegStreamConfigure() once per ffmpegStream instance to setup the areadetector plugin
  • see build_instructions
